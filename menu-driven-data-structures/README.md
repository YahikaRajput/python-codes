# 📌 Python Data Structure Functions Menu Program

## 📖 Description

This project is a menu-driven program in Python that demonstrates various built-in functions of different data structures such as List, Set, Dictionary, String, and Tuple. The program allows users to select a specific data structure and explore its functions interactively through a simple console-based menu system.

## ⚙️ Features

* Interactive menu-based navigation
* Demonstrates List functions like append(), pop(), sort(), etc.
* Covers Set operations like add(), remove(), discard()
* Includes Dictionary methods such as keys(), values(), update()
* Shows String operations like upper(), lower(), capitalize()
* Displays Tuple operations like len(), max(), min(), sum()
* Handles invalid input using exception handling

## 🛠️ Technologies Used

* Python
* Basic programming concepts (loops, functions, conditionals)

## ▶️ How to Run

1. Install Python on your system
2. Copy the code into a `.py` file
3. Run the program using:

   ```bash
   python filename.py
   ```
4. Follow the menu options to explore functions

## 🎯 Purpose

This project is designed for beginners to understand how different data structures work in Python and how their built-in functions can be used in real programs.
